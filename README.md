# Healthcare Web Page

A modern, responsive healthcare web application designed to provide patients with easy access to medical services, doctor information, and appointment booking.

## Features

✨ **Key Features:**
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Service Directory** - Browse all available healthcare services
- **Doctor Profiles** - View information about our qualified medical professionals
- **Easy Appointment Booking** - Simple form to schedule appointments
- **Patient Information** - Access important healthcare information and guidelines
- **Modern UI** - Clean, professional design with intuitive navigation
- **Accessibility** - Built with accessibility standards in mind

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Jay0car/Yolo.git
   cd Yolo
   ```

2. **Open in browser:**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     python -m http.server 8000
     # Then visit http://localhost:8000
     ```

## Usage

### Navigation
- Use the navigation menu to access different sections
- Click on service links to learn more about specific healthcare services
- View doctor profiles to find the right specialist for your needs

### Booking an Appointment
1. Navigate to the "Appointments" section
2. Fill out the appointment booking form with your details
3. Select your preferred date and time
4. Submit the form

### Patient Information
- Visit the "Patient Information" section for guidelines
- Read important health tips and preparation instructions
- Find answers to frequently asked questions

## Customization

### Colors
Edit the color scheme in `styles.css`:
```css
:root {
    --primary-color: #0066cc;
    --secondary-color: #00cc99;
    --text-color: #333;
}
```

### Adding More Sections
1. Add a new `<section>` in `index.html`
2. Add corresponding styles in `styles.css`
3. Update the navigation menu with a link to the new section

### Service List
To add more services, modify the services section in `index.html`
